###Setup
- Make folder “Issue_Tracker” 
- Install python
- Install XAMPP
- Install flask using pip install flask on terminal
- Install mysql using pip install Flask-MySQLdb
- Open XAMPP run apache & mysql & open pypmyadmin using http://localhost/phpmyadmin, make new DB “issue_tracker”
- Make new public repo on git "Issue_Tracker"
    - git init
    - git config --global user.name "username"
    - git config --global user.email “abc123@gmail.com”
- Make py,html,css,js file for basic flask app setup
- Make mysql connection
- Make issues table in phpmyadmin 
- Update issues table in phpmyadmin
- Make add_issue.html file in templates & configure it into app.py to save data into DB
